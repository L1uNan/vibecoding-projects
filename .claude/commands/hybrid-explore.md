Explore the problem space when requirements are unclear.

## Steps

1. Load active change from `openspec/.active` (or ask for one and run `/hybrid-new`).
2. Create `exploration.md` in the active change folder with:
   - problem definition
   - current state
   - candidate approaches
   - research questions
   - risks
3. Update `task_plan.md` (phase `exploration`, progress note).
4. Append a log entry to `progress.md`.
5. Next step: `/hybrid-continue` to create `proposal.md`.
